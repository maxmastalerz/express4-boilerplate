About

SE4B is a express 4.x boilerplate designed with simplicity in mind. It is a fork of atufkas/yanexbo.

This boilerplate utilizes mustacheJS for templating, and is configured to be used with mongoose.

Getting Started

Alter the html files in /views/ and /view/partials

Alter the css files in /public/stylesheets/

Alter your routes in express-routes.js and /routes/
